**<손 동작 추정 기반 CAPTCHA>**



이 프로젝트는 손 동작 추정 기반 CAPTCHA를 만드는 프로젝트입니다.


1. 손 동작 추정 기반 CAPTCHA?

먼저 CAPTCHA는 사람은 쉽지만 봇은 어려운 작업을 통해 사람과 봇을 분류하는 보안 프로그램의 일종입니다. 웹사이트에서 접속하면 볼 수 있는 "로봇이 아닙니다", "신호등을 모두 찾아 클릭하세요" 같은 것들도 CAPTCHA의 일종이라고 할 수 있습니다. 이 프로젝트에서 구현하는 손 동작 추정 기반 CAPTCHA는 기본적으로 카메라로 사용자의 손을 찍고 있는 상태에서 보여주는 손 동작 동영상을 따라하는 방식입니다.
추정인 이유는 손 동작을 일부분 손가림및테스트_handtest1_(2).py 코드를 이용해서 원 형태의 가림막으로 가리기 때문입니다.




2. 실행 방법

   -한번 사용해보기 : prod 파일을 열면(https://mukgore.github.io/handcaptcha/hand_captcha_prod.html) 사용해볼 수 있습니다. 이는 실제 상용화 가능한 CAPTCHA가 아닌, 핵심 CAPTCHA 원리만을 확인하는 일종의 프로토타입입니다. 실행하게 되면 사용자의 카메라를 통해 손 인식을 진행하게 됩니다. 사용 시에는 표시되는 안내에 따르시면 됩니다.  해당 프로그램에서는 총 8가지의 손 동작 영상이 제공되며, 손을 인식시키면 해당 손의 방향(왼손/오른손)에 맞는 손 동작 영상이 표시됩니다. 하나의 동작을 따라하는 과정에서 3번의 사용자 동작 확인 과정을 거칩니다. 해당 과정에서는 하나의 동작 동영상 프레임의 자세와 사용자의 관절 오차가 38도 이하인 상태가 0.2초 이상 지속되는지 확인하고 지속된다면 동작의 다음 부분을 다시 재생하게 됩니다. 최종적으로 각 사용자별로 관절 오차, DTW-관절 오차, Accel 등 3가지의 지표를 가중치를 곱해 합산하여 0~1점 사이의 점수를 가지게 되고, 실제 CAPTCHA에서는 해당 점수를 통해 사람과 봇을 분류하고자 합니다.


   -손 가림 영상 만들어보기 : 손 가림 영상을 직접 만들어보실 수도 있습니다. 손가림및테스트_handtest1_(2).py 코드를 이용하시면 됩니다. 그 코드에서는 손이 나오는 동영상을 입력하면 가림 영상이 출력됩니다. MediaPipe Hands를 통해서 손의 랜드마크 중지 MCP 랜드마크, 손목 랜드마크를 인식하고, 그 중점을 원의 중심으로, 중지 MCP 랜드마크와 손목 랜드마크 사이의 거리를 윈의 지름으로 하는 검정색의 원이 영상의 매 프레임마다 손을 가리게 됩니다.




3. AI와의 기여 분배

저는 이 프로젝트에서 구체적 CAPTCHA의 설계 및 핵심 아이디어를 자연어 및 그림을 통해 표현하여 AI에 입력했습니다. 코드 생성 이후 저는 지속적으로 사용해보며 사용 시 오류의 수정을 요청하며, 사용자 동작 인식 방안, 손을 인식하였을 때 해당 손에 맞춰 자동으로 동작이 제안되는 방안, 손 형태를 반투명하게 표시하는 가이드 방안 등 개선 방안을 제안하였습니다.

AI(claude 웹 서비스, claude code)는 해당 프로젝트 명세사항을 실제 코드로 구현하고 구현되지 않은 사항에 대한 수정을 진행하였습니다. 이외에도 논문 및 선행 서비스 탐색, 인간과 AI의 능력 차이 자료 조사 등에 AI(claude 웹 서비스)를 많이 활용하였습니다.
